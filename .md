# 🖐️ Documentație Gesturi - Hand Battle

Acest document conține definițiile vizuale pentru cele 20 de gesturi utilizate în recunoașterea AI. Fiecare gest trebuie filmat din unghiuri diferite pentru un dataset robust.

## ⚔️ Cărți de Ofensivă

| Gest | Vizual | Descriere Tehnică |
| :--- | :--- | :--- |
| **Atac** | Pistolul Dublu (🔫) | Degetul mare, arătătorul și mijlociul sunt extinse. Inelarul și degetul mic sunt strânse în pumn. |
| **Magie** | Bagheta (🪄) | Degetul arătător și mijlociul sunt ridicate și **lipite** unul de celălalt. Restul degetelor sunt strânse. |
| **Concentrare** | Mudra / Zen (🧘) | Vârful degetului mare atinge vârful inelarului. Arătătorul, mijlociul și degetul mic sunt ridicate și deschise. |
| **Garda_de_Fier** | Pumnul (✊) | Pumn complet strâns. Niciun deget nu iese în afară. |
| **Dubla** | Coarnele (🤘) | Arătătorul și degetul mic sunt ridicate. Mijlociul, inelarul și degetul mare sunt strânse. |

## 🛡️ Cărți de Defensivă

| Gest | Vizual | Descriere Tehnică |
| :--- | :--- | :--- |
| **Scut** | Stop (🖐️) | Toate cele 5 degete sunt ridicate, drepte și depărtate între ele (palma deschisă la maximum). |
| **Bariera** | Spock (🖖) | Toate cele 4 degete lungi sunt ridicate, dar sunt separate la mijloc (arătător+mijlociu lipite, inelar+mic lipite). |
| **Reflectie** | Shaka (🤙) | Degetul mare și degetul mic sunt complet extinse. Cele 3 degete din mijloc sunt strânse în palmă. |
| **Ghimpi** | Degetul Mic | Doar degetul mic (pinky) este ridicat. Celelalte 4 sunt strânse în pumn. |
| **Purificare** | Thumbs Up (👍) | Doar degetul mare este ridicat vertical. Restul degetelor sunt strânse în pumn. |

## 🌀 Cărți de Utilitate

| Gest | Vizual | Descriere Tehnică |
| :--- | :--- | :--- |
| **Buff** | Semnul OK (👌) | Vârful degetului mare atinge vârful arătătorului (formând un cerc). Mijlociul, inelarul și degetul mic sunt ridicate. |
| **Pregatire** | Pinch Mic (🤏) | Vârful degetului mare atinge vârful arătătorului. Celelalte 3 degete sunt strânse complet în palmă. |
| **Debuff** | Thumbs Down (👎) | Degetul mare este extins și arată în jos. Restul degetelor sunt strânse în pumn. |
| **Vulnerabilitate** | Forma L (👆) | Degetul mare și arătătorul sunt extinse la un unghi de 90 de grade. Restul sunt strânse. |
| **Sacrificiu** | Spider-Man (🤟) | Degetul mare, arătătorul și degetul mic sunt extinse. Mijlociul și inelarul sunt strânse. |
| **Adrenalina** | Cercetașul (3️⃣) | Arătătorul, mijlociul și inelarul sunt ridicate. Degetul mic și degetul mare sunt strânse. |
| **Viziune** | **Zen Inelar** (🤚) | **Vârful degetului mare atinge vârful inelarului**. Arătătorul, mijlociul și degetul mic sunt ridicate și deschise. |
| **Arhiva** | Degete Încrucișate (🤞) | Arătătorul și mijlociul sunt ridicate, iar mijlociul este petrecut peste arătător. |
